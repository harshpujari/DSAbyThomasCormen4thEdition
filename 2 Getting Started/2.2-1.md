# 2.2-1

Express the function `n^3 / 1000 + 100 n^2 - 100 n + 3` in terms of Θ-notation.

## Solution

The dominant term in the function is `n^3 / 1000`.

As `n` grows large:
- `n^3 / 1000` grows on the order of `n^3`
- `100 n^2` grows slower than `n^3`
- `-100 n` grows slower than `n^2`
- `3` is constant

Therefore, the function is Θ(`n^3`).

## Justification

For large enough `n`, the cubic term dominates all lower-order terms and constants.
We can write:

`n^3 / 1000 + 100 n^2 - 100 n + 3 = Θ(n^3)`

because there exist positive constants `c1`, `c2`, and `n0` such that for all `n >= n0`:

`c1 n^3 <= n^3 / 1000 + 100 n^2 - 100 n + 3 <= c2 n^3`

One possible choice is:
- `c1 = 1/2000`
- `c2 = 2`
- `n0 = 1000`

This shows the function grows at the same rate as `n^3` asymptotically.
